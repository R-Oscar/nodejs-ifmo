## API Yahoo.Weather

Запуск:

```bash
$ node index.js
```

Скрипт подключается к [Yahoo.Weather API](https://developer.yahoo.com/weather/?guccounter=1) и считывает информацию о погоде в городе Санкт-Петербург на завтрашний день (нижняя граница температуры)
